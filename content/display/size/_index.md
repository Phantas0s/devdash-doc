DevDash is based on a 12 columns grid.

You can indicate the width of a widget in number of column (1 to 12), or using the equivalent t-shirt size as described below:

| Name   | Number of columns   |
| ------ | ------------------- |
| xxs    | 1                   |
| xs     | 2                   |
| s      | 4                   |
| m      | 6                   |
| l      | 8                   |
| xl     | 10                  |
| xxl    | 12                  |
